# particle-editor

# About
Editor for GlyphX's particle systems


This is entirely based off of Mike.NL's work, with edits made by Claude for quality of life features and attempts to fix bugs. See the original repo here: https://github.com/GlyphXTools/particle-editor

# Usage
On the x64 build, launching with no flag opens the new WebView2/React UI (the
default). Pass `--legacy` (or its alias `--legacy-ui`) to opt back into the
classic Win32 chrome. The x86 build is always legacy. (`--new-ui` is accepted
but redundant — it now selects the default.)
